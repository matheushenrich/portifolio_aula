CONHECIMENTOS

INICIO

HTML

Possuo conhecimentos básicos em HTML, permitindo a criação da estrutura de páginas web organizadas e semânticas. Sei utilizar corretamente as principais tags para formatação de texto, criação de listas, tabelas, formulários e inclusão de elementos multimídia, como imagens e vídeos.

PYTHON

Tenho experiência introdutória com Python, compreendendo a sintaxe da linguagem e a criação de scripts básicos. Também sei trabalhar com bibliotecas básicas para automação de tarefas e análise de dados simples.

CSS

Tenho noções básicas de CSS para estilizar páginas web, aplicando cores, fontes, espaçamentos e layouts responsivos. Sei utilizar seletores, classes e IDs, além de propriedades como flexbox e grid para criar layouts modernos e adaptáveis a diferentes dispositivos.

SQL

Entendo os princípios básicos do SQL e seu uso em bancos de dados relacionais. Tenho conhecimento na criação de consultas simples para inserir, buscar, atualizar e deletar informações. Sei utilizar comandos como SELECT, INSERT, UPDATE e DELETE, além de realizar junções básicas entre tabelas.
